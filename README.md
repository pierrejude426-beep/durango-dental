# Durango Dental Group - Insurance Coordinator

Live demo dashboard built by Phare. Reads claim, eligibility, denial, and A/R data from a Supabase backend in real time.

Live: https://durango-dental-group.vercel.app

- Single static page (index.html), no build step
- Data layer: Supabase RPCs (dental_*)
- Patient ledger is simulated; practice profile is from public sources
